# WEB-ONG

Diseño estático de una web para una ONG inventada. Incluye páginas informativas sobre la organización, proyectos, noticias, participación y donaciones, junto con recursos de estilo e imágenes.

## Contenido

- `index.html`: portada principal.
- `proyectos.html` y `proyecto.html`: listado y detalle de proyectos.
- `noticias.html` y `noticia.html`: listado y detalle de noticias.
- `participar.html`: información para colaborar.
- `donar.html`: información para donar.
- `faq.html`: preguntas frecuentes.
- `CSS/`, `JS/`, `IMG/`: estilos, scripts e imágenes.

## Uso

Abre `index.html` en un navegador para ver el diseño.
